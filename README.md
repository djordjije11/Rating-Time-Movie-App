# Rating Time - Movie App

The app enables users to search for any movie available in The Movie Database (TMDB) API on the home page and rate it according to their preference. Users can also filter movies by genre while going through pages of movies list. On the Rated Movies page, users can view all their ratings for movies and delete them if needed. Users can register and log in with their registered accounts. Users with the Admin role, in addition to the previously mentioned functionalities, also have access to the Users page, where all application users are listed. By selecting a specific user, an admin can view their movie ratings.

## Implementation

The application logic is developed using the C# ASP.NET framework along with the EntityFramework, used for accessing and manipulating data in the database, while the user interface is created using the React.js library. The application's authentication relies on the HTTP cookies mechanism.
